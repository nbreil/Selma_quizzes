🫁 Activité 2 – La respiration

Trajet de l'air dans le corps

Nez / Bouche → Trachée → Bronches → Bronchioles → Alvéoles pulmonaires

Les alvéoles pulmonaires sont entourées de vaisseaux sanguins. C'est là que se font les échanges entre l'air et le sang.

Le rôle du diaphragme

Le diaphragme est un muscle situé sous les poumons.

  • Il se contracte et s'abaisse → les poumons se gonflent → c'est l'inspiration (l'air entre).
  • Il se relâche → les poumons se dégonflent → c'est l'expiration (l'air sort).

Les gaz échangés

  • L'organisme a besoin de dioxygène (O₂) pour fonctionner.
  • Les organes produisent un déchet gazeux : le dioxyde de carbone (CO₂).
  • Le CO₂ est transporté par le sang jusqu'aux poumons, puis rejeté lors de l'expiration.

Pourquoi respire-t-on plus vite à l'effort ?

Plus les organes travaillent → plus ils ont besoin de dioxygène → la respiration s'accélère.
❤️ Activité 3 – La circulation sanguine

Le rôle du sang

Le sang transporte :

  • Le dioxygène (O₂) des poumons vers les organes
  • Le dioxyde de carbone (CO₂) des organes vers les poumons
  • Les nutriments vers les organes

Le cœur et le pouls

Le cœur est un muscle qui fonctionne comme une pompe.

  • En se contractant, il envoie le sang dans les artères.
  • Le sang revient au cœur par les veines.
  • Le pouls = battements du cœur que l'on sent dans les vaisseaux.
  • Le rythme cardiaque s'accélère quand on court ou quand on a peur.

Artères vs Veines – le moyen de s'en souvenir

💡 Artère = Aller (du cœur vers les organes)
💡 Veine = reVenir (des organes vers le cœur)

Les deux circuits du sang

Circulation générale (cœur ↔ organes du corps) :

Cœur → Artères → Organes du corps → Veines → Cœur

Circulation pulmonaire (cœur ↔ poumons) :

Cœur → Artères pulmonaires → Poumons → Veines pulmonaires → Cœur
